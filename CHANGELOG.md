# Changelog

Все значимые изменения в проекте документируются в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

## [1.0.0] - 2024-12-30

### Добавлено

- Первый публичный релиз AVYX Beta
- Главный экран с лентой заданий и шоукейсом работ
- Поиск заказов с фильтрами по категории, бюджету и сложности
- Профиль пользователя с системой уровней и достижений
- Система миссий и квестов
- Ежедневные спринты с голосованием
- Маркетплейс с PRO-статусом
- AI-анализ заданий (в разработке)
- "Сад достижений" — визуализация прогресса
- Система защищенных сделок (эскроу)
- Адаптивный мобильный интерфейс
- Онбординг для новых пользователей
- Создание заказов с AI-генерацией описания
- Система уведомлений
- Редактирование профиля с кастомизацией

### Технические изменения

- Миграция на React 19
- Настройка Vite для сборки
- TypeScript strict mode
- Tailwind CSS для стилизации
- Структурирование проекта по папкам
